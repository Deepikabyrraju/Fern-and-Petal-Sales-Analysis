# Ferns & Petals Sales Analysis (Excel)

This project analyzes the sales performance of Ferns & Petals using Excel.  
It includes data cleaning, exploratory analysis, and an interactive dashboard.

## Objectives
- Understand overall sales trends
- Identify top-performing products and categories
- Analyze sales by city/channel
- Find monthly/seasonal patterns in orders

## Steps Performed
1. **Data Cleaning**
   - Removed duplicates and blanks
   - Fixed date formats
   - Handled incorrect values

2. **Data Preparation**
   - Created new columns (e.g., Month, Day, Order Type)
   - Used formulas for metrics like total revenue, quantity, etc.

3. **Analysis**
   - Pivot tables for:
     - Sales by city
     - Sales by product category
     - Monthly sales trend
   - Slicers for filtering by:
     - City
     - Category
     - Month/Year

4. **Dashboard**
   - Visuals: bar charts, line charts, cards
   - KPIs: total sales, total orders, average order value

## Tools Used
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - Power Query
  - Power Pivot / Data Model 

## How to View
- Download the Excel file `Ferns_and_Petals_Analysis.xlsx`
- Open in Excel to explore the pivot tables and dashboard
